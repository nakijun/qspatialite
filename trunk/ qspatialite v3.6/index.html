QSpatialite Plugin

Description

QSpatiaLite is a QGIS python plugin to manage SpatiaLite databases within QGIS:

- Basic DataBase management
- QGIS Integration: import QGIS layers, View Spatial Tables and Queries in QGIS
- SQLeditor: Basic SQL editor with highlighting and autocompletion
- SQL Query Builder: Build queries easily, like in MapInfo + Unique values for columns + Use Spatial Index
